pyEnigma
============

#### Python implementation of the Enigma machine.


Usage
-----

    $ echo "Hello World" | python3.2 cypher_enigma_safe.py secret B  I II V 
    Vjpfw Ekdxj

    $ echo "Vjpfw Ekdxj" | python3.2 cypher_enigma_safe.py secret B  I II V
    Hello World
